# Basic SQL Commands
* Select
```
  SELECT * FROM <TABLE_NAME> WHERE <CONDITION>
```
* Delete
```
  DELETE FROM <TABLE_NAME> WHERE <CONDITION>
```
* Update
```
  UPDATE <TABLE_NAME> SET <COLUMN NAME> = <VALUE> WHERE <CONDITION>
```
* Insert
```
  INSERT INTO <TABLE_NAME> (<COLUMN?NAME>) VALUES (<VALUES>)
```
